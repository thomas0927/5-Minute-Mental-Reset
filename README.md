# 5-Minute Mental Reset — Claude Code Skills

Claude Code skills based on *5-Minute Mental Reset* by Thomas Bright.

## Installation

In Claude Code:

`/plugin marketplace add thomas0927/5-Minute-Mental-Reset`
`/plugin install 5-minute-mental-reset`

That's it — Claude Code will fetch the repo and register all skills automatically.

Alternative: install from a local clone

1. `git clone https://github.com/thomas0927/5-Minute-Mental-Reset`
2. `cd 5-Minute-Mental-Reset`
3. `/plugin marketplace add /absolute/path/to/5-Minute-Mental-Reset`
4. `/plugin install 5-minute-mental-reset`

## The 5-Minute Mental Reset Journey

The skills follow the book's progression:

Reframe — Anxiety is a signal, not a defect
Understand — Why methods fail under stress
Reset — Five-minute tools that work in real time
Rescue — Scenario playbooks for high-pressure moments
Immunity — Prevention, automation, and teaching
